## Summary
My barebones NeoVIM configuration.

## Quickstart
> Unix, Linux, Wsl Installation

```shell
git clone --depth 1 https://github.com/grmisasi/nvim-config-lua ~/.config/nvim/
chmod u+x ~/.config/nvim/INSTALL.sh
./INSTALL.sh
```

## Uninstall
> Unix, Linux, Wsl Installation

```shell
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim/site/pack
```
